## Install

```bash
yarn add vuepress-theme-modern-blog -D
# OR npm install vuepress-theme-modern-blog -D
```

## Usage

```js
// .vuepress/config.js
module.exports = {
  theme: 'modern-blog',
  themeConfig: {
    // for the full list of options read more in [Docs](./docs/theme-config.md)
  }
}
```
